# Проект Base BD

## Описание:

Проект Base BD - это поиск вакансий с подключением БД. Позволяет получить данные о компаниях и вакансиях с сайта hh.ru, 
спроектировать таблицы в БД PostgreSQL и загрузить полученные данные в созданные таблицы.

## Установка:

1. Клонируйте репозиторий:
```
git@github.com:pannikolaserg-prog/Base-BD.git
```

## Использование:

1. Откройте Base BD в вашем PyCharm.
2. Откройте модуль main и следуйте указаниям в терминале.
3. Модуль DB_Manager обеспечивает взанимодействие с базой данных и позволяет создавать необходимые таблицы.
4. Модуль API_Connect позволяет подключиться с API сайта HH.ru.
